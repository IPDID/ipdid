# Usage
<!-- usage -->
```sh-session
$ npm install -g ipdid
$ ipdid COMMAND
running command...
$ ipdid (-v|--version|version)
ipdid/0.0.1 darwin-arm64 node-v16.0.0
$ ipdid --help [COMMAND]
USAGE
  $ ipdid COMMAND
...
```
